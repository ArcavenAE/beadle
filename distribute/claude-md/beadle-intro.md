## beadle (issue triage + dashboard)

This repo is a beadle target. beadle triages issues against this project's
declared intent (`.beadle/intent.yaml`), weighted by what maintainers actually
act on, and maintains a pinned dashboard issue (`📋 beadle — Triage Dashboard`)
whose state lives out-of-band. Run a pass via the `beadle-triage` skill. beadle
proposes; it does not authorize work or auto-close issues. See
`.claude/rules/beadle-invariants.md`.
